# QA-java-diplom-3

This project was completed as part of the Yandex Practicum diplom project.
The project includes automated UI tests in Java using Selenium.

### Run the project

- Go to the project folder
- Launch the project
```
mvn clean test
```

### Requirements
| Dependency | Version |
| ------------ | ------------ |
| JDK | 11.0.26  |
